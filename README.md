# 🔋 Battery Sales EDA Dashboard

A Streamlit-based Battery Sales Analytics Dashboard for Exploratory Data Analysis (EDA).

## Features

- Sales Analysis
- Product Analysis
- Zone Analysis
- Regional Manager Performance
- Customer Insights
- Payment Analysis
- Return Analysis
- Correlation Heatmap
- Interactive Dashboard

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
